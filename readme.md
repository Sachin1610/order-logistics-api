# Order & Logistics API

Node.js + Express + MongoDB (Atlas) API for creating and viewing orders.

## Features
- Connects to MongoDB Atlas using Mongoose
- Create Order: `POST /orders`
- View Orders: `GET /orders`
- Health check: `GET /`

## Tech Stack
- Node.js, Express
- MongoDB Atlas, Mongoose
- dotenv
